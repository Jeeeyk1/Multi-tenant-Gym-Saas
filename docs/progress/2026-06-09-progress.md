## Key Decisions

- **`@IsEnum()` over `@IsIn()`**: Enums are now the source of truth for valid values. Adding a new value means updating the enum only — the DTO validation follows automatically.
- **`AI_BASE_URL` env var**: External service URLs must never be hardcoded. Both providers receive the base URL from `ConfigService`, defaulting to the correct public endpoints. This allows staging environments to point at proxies without code changes.
- **No equipment question**: Removed from onboarding — all members have gym access (assumption in this context).
- **Target weight on step 3**: Placed immediately after current weight so the two are visually adjacent in the flow.
- **`profileToForm` helper**: Converts `MemberProfile | null` to the edit form state, centralising all null→empty-string and null→null conversions.
