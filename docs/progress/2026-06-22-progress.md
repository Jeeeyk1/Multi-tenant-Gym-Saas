## Key decisions

- `upsertMembership` is non-nullable (Prisma upsert always returns a record). The `NOT_CONVERSATION_MEMBER` guard is only reachable for non-COMMUNITY conversation types where `findMembership` (nullable) is called. Tests that exercise this path must use `type: 'DIRECT'` or another non-COMMUNITY type.
